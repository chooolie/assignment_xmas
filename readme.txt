Menu will show when game is started.
In the menu you can either choose to play or choose difficulty.
If you play it will begin in level easy.
If you decide to chose difficulty, you click on one, then click play.
When playing you use 
	-"w" key for going up 
	-"s" key for going down
	-"d" key for going right
	-"a" key for going left
You are controlling the blue ball.
You must cross to the other side
If you hit a pink ball it brings you back to start.
You have 10 lifes
If you run out of lives, game over, return to menu
If you get past the level, level 2 will commence
You again must get to the other side
If you succeed then yo win over all
If you dont succeed in 10 lives, then game over.
